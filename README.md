# HBC PWA Project

1. Fill your website url to MAGENTO_BACKEND_URL and braintree public key to BRAINTREE_TOKEN at packages/siminia/.env

2. Update the port you want to use to run NodeJs and change https://staging.hairbowcenter.com/ to your magento URL at start_pwa_with_build.sh

3. Run yarn install

4. Run start_pwa_with_build.sh
